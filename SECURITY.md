# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Vulnerabilities can be reported as a [BUG] in [issues](https://github.com/majusko/pulsar-java-spring-boot-starter/issues) section.
